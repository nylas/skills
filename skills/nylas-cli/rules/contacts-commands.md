## Contacts Commands

```bash
nylas contacts list                           # List contacts
nylas contacts show <id>                      # Show contact
nylas contacts create --first-name N --last-name L --email E
nylas contacts search --email user@example.com # Search contacts
nylas contacts sync                           # Show how v3 contact sync works
nylas contacts groups list                    # List groups
```
