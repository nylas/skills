## Audit Logging Commands

```bash
nylas audit init                              # Initialize audit logging
nylas audit logs enable                       # Enable logging
nylas audit logs disable                      # Disable logging
nylas audit logs status                       # Check logging status
nylas audit logs show                         # View logs
nylas audit logs summary                      # View log summary
nylas audit logs clear                        # Clear logs
nylas audit export                            # Export audit logs
nylas audit config show                       # Show audit config
nylas audit config set <key> <value>          # Set audit config
```
