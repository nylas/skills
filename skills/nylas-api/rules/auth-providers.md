## Supported Providers & Setup

| Provider | Auth Type | Setup Guide |
|----------|-----------|-------------|
| Google (Gmail, Workspace) | OAuth 2.0 | https://developer.nylas.com/docs/provider-guides/google/create-google-app/ |
| Microsoft (Outlook, 365) | OAuth 2.0 | https://developer.nylas.com/docs/provider-guides/microsoft/create-azure-app/ |
| Yahoo | OAuth 2.0 | https://developer.nylas.com/docs/provider-guides/yahoo-authentication/ |
| iCloud | App password | https://developer.nylas.com/docs/provider-guides/icloud/ |
| IMAP | Username/password | https://developer.nylas.com/docs/provider-guides/imap/ |
| Exchange (EWS) | Username/password | https://developer.nylas.com/docs/provider-guides/exchange-on-prem/ |
| Zoom (meetings) | OAuth 2.0 | https://developer.nylas.com/docs/provider-guides/zoom-meetings/ |

### Provider-Specific Notes

**Google:**
- Requires Google Cloud project with OAuth consent screen
- Workspace service accounts: https://developer.nylas.com/docs/provider-guides/google/google-workspace-service-accounts/
- Shared GCP app: https://developer.nylas.com/docs/provider-guides/google/shared-gcp-app/
- Google Pub/Sub: https://developer.nylas.com/docs/provider-guides/google/connect-google-pub-sub/
- Verification guide: https://developer.nylas.com/docs/provider-guides/google/google-verification-security-assessment-guide/

**Microsoft:**
- Requires Azure AD app registration
- Admin approval: https://developer.nylas.com/docs/provider-guides/microsoft/admin-approval/
- Shared folders: https://developer.nylas.com/docs/provider-guides/microsoft/shared-folders/
- Troubleshooting: https://developer.nylas.com/docs/provider-guides/microsoft/troubleshooting/

**Detailed provider setup steps live in the linked guides above.**

Reference: [All provider guides](https://developer.nylas.com/docs/provider-guides/)
